# 2024 QB Efficiency Analysis

This project analyzes 2024 NFL quarterback performance using weekly player data aggregated to season totals.

## Methodology

- Filtered to 2024 regular season
- Aggregated weekly data to season totals
- Required minimum 200 pass attempts
- Calculated:
  - Yards per Attempt
  - TD Rate
  - INT Rate
  - EPA per Attempt

## Key Finding

Lamar Jackson was a clear efficiency outlier, leading all qualified quarterbacks in EPA per attempt

## Tools Used

- Python
- Pandas
- Matplotlib
